# ระบบเลือกกิจกรรมชุมนุมออนไลน์

ระบบจัดการและเลือกกิจกรรมชุมนุมสำหรับนักเรียนโรงเรียนแก่นนครวิทยาลัย

## คุณสมบัติหลัก

- ✅ **ระบบจัดการนักเรียน** - เพิ่ม/แก้ไข/ลบข้อมูลนักเรียน
- ✅ **ระบบจัดการชุมนุม** - สร้างและจัดการชุมนุมต่างๆ
- ✅ **ระบบจัดการครู** - จัดการข้อมูลครูที่ปรึกษา
- ✅ **การเลือกชุมนุมออนไลน์** - นักเรียนสามารถเลือกชุมนุมผ่านเว็บไซต์
- ✅ **ระบบนำเข้าข้อมูล CSV** - อัพโหลดข้อมูลจาก Excel/CSV
- ✅ **ระบบรายงาน** - ส่งออกรายชื่อสมาชิกชุมนุม
- ✅ **ระบบล็อกชุมนุม** - ปิด/เปิดการรับสมัครชุมนุมแต่ละชุมนุม
- ✅ **ระบบกรองตามระดับชั้น** - ชุมนุมสามารถกำหนดระดับชั้นที่รับได้

## เทคโนโลยีที่ใช้

- **Frontend**: HTML5, CSS3, Bootstrap 5, JavaScript
- **Backend**: PHP 7.4+
- **Database**: MySQL/MariaDB
- **Icons**: Font Awesome 6
- **Framework**: Vanilla PHP (ไม่ใช้ Framework)

## การติดตั้ง

### ความต้องการของระบบ
- PHP 7.4 หรือสูงกว่า
- MySQL 5.7 หรือ MariaDB 10.2+
- Web Server (Apache/Nginx)
- Extension: mysqli, session

### ขั้นตอนการติดตั้ง

1. **Clone repository**
```bash
git clone https://github.com/krujakkr/club.git
cd club
```

2. **ตั้งค่าฐานข้อมูล**
- สร้างฐานข้อมูลใหม่ใน MySQL
- นำเข้าไฟล์ `database/club_system.sql`

3. **ตั้งค่าการเชื่อมต่อฐานข้อมูล**
แก้ไขไฟล์ `config.php`:
```php
define('DB_SERVER', 'localhost');
define('DB_USERNAME', 'your_username');
define('DB_PASSWORD', 'your_password');
define('DB_NAME', 'your_database_name');
```

4. **สร้างบัญชีผู้ดูแลระบบ**
รันไฟล์ `fix_admin.php` เพื่อสร้างบัญชี admin:
- Username: `admin`
- Password: `admin123`

## การใช้งาน

### สำหรับผู้ดูแลระบบ
1. เข้าสู่ระบบที่ `/admin_login.php`
2. จัดการข้อมูลครู -> ชุมนุม -> นักเรียน ตามลำดับ
3. เปิด/ปิดระบบลงทะเบียน
4. ดูรายงานและส่งออกข้อมูล

### สำหรับนักเรียน
1. เข้าเว็บไซต์หลัก
2. กรอกเลขบัตรประชาชน 13 หลัก + รหัสนักเรียน
3. เลือกชุมนุมที่สนใจ
4. ยืนยันการเลือก

## โครงสร้างไฟล์

club/

├── admin.php               # หน้าจัดการนักเรียน

├── admin_login.php         # หน้าเข้าสู่ระบบแอดมิน
├── student_login.php       # ระบบเข้าสู่ระบบนักเรียน
├── select_club.php         # ระบบเลือกชุมนุม
├── import_csv.php          # นำเข้าข้อมูล CSV
├── export_csv.php          # ส่งออกข้อมูล CSV

## ฟีเจอร์เด่น

### 🎯 ระบบจัดการชุมนุมที่ยืดหยุ่น
- กำหนดจำนวนสมาชิกสูงสุดต่อชุมนุม

- ล็อก/ปลดล็อกชุมนุมเมื่อไม่ต้องการรับสมัคร
